# Java Projects - Estudo e Evolução

Este repositório centraliza os meus projetos desenvolvidos em Java, focando na transição da lógica de programação para a **Orientação a Objetos (POO)**. O objetivo é documentar a aplicação prática de conceitos aprendidos durante o curso de ADS e estudos autônomos.

---

## Estrutura do Repositório

O repositório está organizado por módulos de projeto. Cada pasta possui o seu próprio código-fonte e documentação específica:

### [Sistema de Gestão de Estoque](./Estoque)
* **Conceitos:** Classes, Atributos, Métodos, Construtores e Encapsulamento.
* **Descrição:** Um sistema simples para gerenciar a entrada e saída de produtos, calculando o valor total em stock com formatação de dados.

### [Sistema Bancário](./SistemaBancario)
* **Conceitos:** Sobrecarga de Construtores, Encapsulamento Rigoroso e Lógica de Negócio.
* **Descrição:** Simulação de operações de conta bancária (depósito e levantamento) com aplicação de taxas de serviço e proteção de atributos sensíveis como o saldo.

## Tecnologias e Ferramentas
- **Linguagem:** Java (JDK 17+)
- **IDE:** Eclipse
- **Controlo de Versão:** Git & GitHub

## Objetivos de Aprendizagem
Atualmente, os projetos aqui presentes demonstram o domínio de:
1.  **Encapsulamento:** Proteção de dados com modificadores `private` e uso de *Getters/Setters*.
2.  **Sobrecarga:** Criação de múltiplos construtores para flexibilidade na instanciação de objetos.
3.  **Lógica de Negócio:** Centralização de regras (como taxas e cálculos) dentro das entidades.
4.  **Boas Práticas:** Organização de pacotes (`entities` e `application`) e nomes de variáveis claros.
