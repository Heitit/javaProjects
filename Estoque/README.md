# Projeto: Sistema de Gestão de Estoque

Este módulo faz parte da minha trilha de aprendizado em Java e foca na aplicação de **Orientação a Objetos**.

### Conceitos Aplicados neste Projeto:
- **Encapsulamento:** Uso de modificadores de acesso (private) e métodos Getters/Setters.
- **Construtores:** Inicialização de objetos com dados obrigatórios.
- **Sobrecarga:** Diferentes formas de instanciar um produto.
- **Lógica de Negócio:** Métodos para adicionar e remover quantidades do estoque.

### Estrutura:
- `Application/Project.java`: Interface via console para interação com o usuário.
- `Entities/Product.java`: Classe que representa a entidade Produto e suas regras.
